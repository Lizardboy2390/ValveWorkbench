◆ eTriodeParameter

enum eTriodeParameter

The eTriodeParameter enum.

Defines indexes into the array of model Parameters. This is useful because the models considered represent progressive refinements, i.e. the more complex models reuse parameters from the simpler models and complex models only need to define the additional parameters that they add to the model
